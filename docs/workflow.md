# Workflow — Quy trình 9 cột cho 1 tập podcast

Board GitHub Projects (kanban) với 9 cột:

1. **Ideation** — lên ý tưởng, chọn chủ đề, khách mời.
2. **Scripting** — viết kịch bản, câu hỏi, outline.
3. **Recording** — quay/ghi âm. Link Drive để ở Issue.
4. **Editing** — dựng, làm sạch, mix nhạc.
5. **Packaging** — thumbnail, title A/B, description, show notes, tags.
6. **Scheduling** — lên lịch phát hành các nền tảng.
7. **Published** — đã đăng. Dán link YouTube/Spotify/... vào Issue.
8. **Repurpose** — tạo Issue [CLIP] cắt đoạn ngắn cho TikTok/Reels.
9. **Analytics** — sau 7 ngày ghi sub/view/retention/engagement.

## Labels chuẩn
Xem `labels.md`. Mỗi Issue gắn: 1 nền tảng + 1 định dạng + 1 trạng thái + 1 vai trò.

## Tự động hóa gợi ý
- Claude Code + ccpm: sinh kế hoạch tháng.
- Make/Zapier: khi cột Published xong → tự đẩy clip sang TikTok/Reels.
- Cross-platform conversion theo dõi ở cột Analytics.
